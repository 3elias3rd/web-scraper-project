# Weather API webscraper

Scrape OpenWeatherMap to return the current or future weather for the location that is to be decided by the user.

## How it works

System promts the user for a location and the forecast they are interested in. If the user is interested in todays forcast for dubai, the system collects the information, today and Dubai and return the weather forecast details.

Project is currently centered around one file, 'main.py'.
